# GitHub Pages

Этот каталог готов для публикации как отдельный сайт-визитка.

## Как включить

1. В репозитории откройте `Settings -> Pages`.
2. В `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
3. Сохраните настройки.

После этого сайт будет работать отдельно от Telegram-бота.
